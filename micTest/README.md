# Audio Artifact Detector

Скрипт для детекции потерянных пакетов и артефактов в аудиопотоке по сети.

## Принцип работы

1. Генерируется тестовый сигнал (синус или меандр) на аудиовыход по умолчанию
2. Одновременно записывается вход с микрофона (к которому подключено устройство, получающее звук по сети)
3. Входной сигнал анализируется в реальном времени на наличие аномалий
4. Все обнаруженные аномалии логируются в файл с временными метками

## Детектируемые аномалии

| Тип | Описание |
|-----|----------|
| **DROPOUT** | Пропадание сигнала (RMS ниже порога) |
| **CLICK** | Резкие скачки амплитуды (щелчки) |
| **LEVEL_DROP** | Значительное падение уровня сигнала |
| **LEVEL_SPIKE** | Значительный всплеск уровня сигнала |
| **FREQ_DRIFT** | Отклонение частоты от ожидаемой |
| **DISTORTION** | Гармонические искажения (THD) |

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
# Базовый запуск: 5 минут, синус 1000 Гц
python audio_artifact_detector.py 5

# 10 минут, меандр
python audio_artifact_detector.py 10 --wave square

# 2 минуты, частота 440 Гц
python audio_artifact_detector.py 2 --freq 440

# Без сохранения WAV записи
python audio_artifact_detector.py 5 --no-save

# Список аудиоустройств
python audio_artifact_detector.py 0 --list-devices
```

## Аргументы

| Аргумент | Описание | По умолчанию |
|----------|----------|-------------|
| `duration` | Длительность теста (минуты) | обязательный |
| `--freq` | Частота тестового сигнала (Гц) | 1000 |
| `--wave` | Тип волны: `sine` / `square` | sine |
| `--sample-rate` | Частота дискретизации | 44100 |
| `--block-size` | Размер блока обработки | 1024 |
| `--dropout-threshold` | Порог RMS для dropout | 0.02 |
| `--click-threshold` | Множитель для кликов | 5.0 |
| `--thd-threshold` | Порог THD для искажений | 0.15 |
| `--no-save` | Не сохранять WAV запись | false |
| `--list-devices` | Показать аудиоустройства | — |

## Выходные файлы

- `artifact_log_YYYYMMDD_HHMMSS.txt` — лог аномалий с временными метками
- `recording_YYYYMMDD_HHMMSS.wav` — запись с микрофона (если не `--no-save`)

## Формат лога

```
[2026-03-16T12:00:01.234] T+  15.320s | DROPOUT        | длительность:    23.22 мс | severity: 0.232 | RMS упал ниже 0.020
[2026-03-16T12:00:05.678] T+  20.150s | CLICK          | длительность:     0.02 мс | severity: 0.850 | Скачок амплитуды: 0.4521
```
